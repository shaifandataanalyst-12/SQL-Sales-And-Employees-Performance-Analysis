# SQL Sales & Employee Performance Analysis

## Project Overview
This project demonstrates advanced SQL analysis on a sales and employee dataset to extract business insights related to employee performance, sales trends, regional performance, and Business revenue

The project solves **30 business-driven SQL analytical questions** using joins, window functions, aggregations, ranking functions, and CTEs.

## Tools & Technologies
- SQL (MySQL)
- Relational Database Concepts
- Joins
- Window Functions
- Aggregations
- CTEs

## Dataset
The project uses two relational tables:

### Employees Table
- employee_id
- first_name
- last_name
- department
- gender
- location
- date_of_joining

### Sales Table
- sale_id
- employee_id
- product
- units_sold
- unit_price
- total_sale
- discount
- payment_method
- region
- sale_date

- LEFT JOIN

## Business Questions Solved

Examples of insights extracted:

• Identified employees generating **₹100K+ sales in 2023**

• Found employees working in **multiple regions**

• Ranked **top 3 sales performers per region**

• Categorized employees as **High, Medium, Low performers**

• Calculated **department-wise contribution to total revenue**

• Identified **employees selling across all product categories**

• Analyzed **payment method distribution**

• Evaluated **discount impact on revenue**

## Key Insights

- Top-performing employees significantly contribute to overall company revenue.
- Certain regions consistently generate higher sales volumes.
- UPI and Net Banking payments dominate over cash transactions.
- High discount strategies may reduce profitability.
